## 🔍 Project Overview
Analyzed **39,000+ real U.S. government records** from the Health Resources & Services Administration (HRSA) to identify healthcare workforce shortage patterns across all 50 states.

## 💡 Key Findings
- **New York** has the highest total underserved population (76M+)
- **Kentucky & Wyoming** have the worst shortage severity scores
- **Non-Rural areas** surprisingly have more shortage designations than Rural
- **260M+ Americans** are affected by Primary Care shortages
